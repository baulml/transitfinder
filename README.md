### Requirements
  * Python 3
  * Numpy
  * Astropy
  * Matplotlib
  * Tkinter
### Installation
```python
pip3 install numpy
pip3 install astropy
pip3 install matplotlib
pip3 install tkinter
```
### Usage
```python
python3 transit.py
```
